# datepicker
通过原生HTML/CSS/JavaScript完成一个日期选择器



## 日期对象
获取年： getFullYear()

获取月份： getMonth()  

获取日期： getDate()

获取星期： getDay() =>[1,2,3,4,5,6,0]

当月的第一天：new Date(year,month-1,1)

当月的最后一天： new Date(year,month,0)


## 课程总结
1.使用HTML合理规划组件结构

2.为组件编写美观的样式

3.如何使用JavaScript获取组件所需的数据

4.将数据与HTML结合

5.用户事件处理
